# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it via email at **kazkozdev@gmail.com**. Do not open a public GitHub issue for security vulnerabilities.

You should receive a response within 48 hours. Please include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
